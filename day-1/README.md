# Day 1 — Git, APIs, and the Developer Mindset

## Projects

### cli-task-manager
A command-line task manager built with Node.js. Covers CLI tool design, file-based persistence, and formatting terminal output with color and tables.

### bookstore-api
A REST API for a bookstore backed by SQLite. Covers Express.js, CRUD endpoints, input validation middleware, and working with a local database.

## How to work on these
1. Open the project folder you want to build
2. Read `PROMPTS.md`
3. Create a new empty folder on your machine, open Claude Code inside it
4. Follow the prompts to build the project
5. Compare your result with `solution/`
