# Day 3 — Agentic Engineering

## Projects

### quicknote-extension
Build a Chrome extension from scratch. Covers Manifest V3 architecture, service workers, content scripts, chrome.storage, and context menus. Built entirely with Claude Code from a plain-English description.

### habitflow-saas
Build a full habit tracking web app using PRD-driven development. Instead of writing code directly, you write (and read) a detailed Product Requirements Document and let Claude Code build the entire application from it. Covers Next.js 14, SQLite, Tailwind CSS, and streak calculation logic.

## How to work on these
1. Open the project folder you want to build
2. Read `PROMPTS.md`
3. Create a new empty folder on your machine, open Claude Code inside it
4. Follow the prompts to build the project
5. Compare your result with `solution/`
