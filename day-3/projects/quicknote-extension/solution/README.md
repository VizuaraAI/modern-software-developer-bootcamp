# QuickNote -- Chrome Extension

A Chrome Extension that lets you highlight text on any web page and save it as a note. Built as a workshop demo to show how AI can build a complete, functional Chrome extension from scratch.

## What It Does

- **Right-click to save**: Highlight any text on a webpage, right-click, and select "Save to QuickNote"
- **Toast notification**: A brief "Note saved!" toast appears on the page confirming the save
- **Browse your notes**: Click the extension icon to see all saved notes in a clean popup
- **Search notes**: Filter your saved notes by text, page title, or URL
- **Delete notes**: Remove individual notes or clear all notes at once
- **Persistent storage**: Notes are saved to Chrome's local storage and persist across browser restarts

## Installation

### Step 1: Generate the icons

The extension needs icon files. Run the icon generator script:

```bash
cd quicknote-extension
node create-icons.js
```

This creates `icon48.png` and `icon128.png` in the extension directory.

### Step 2: Load the extension in Chrome

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **"Load unpacked"**
4. Select the `quicknote-extension` folder
5. The QuickNote extension icon should appear in your toolbar

If the icon does not appear in the toolbar, click the puzzle piece icon in the Chrome toolbar and pin QuickNote.

### Step 3: Use it

1. Go to any web page
2. Highlight some text
3. Right-click and select **"Save to QuickNote"**
4. A toast notification confirms the save
5. Click the QuickNote extension icon to see your saved notes

## Files Overview

| File              | Purpose                                               |
|-------------------|-------------------------------------------------------|
| `manifest.json`   | Chrome extension configuration (Manifest V3)          |
| `background.js`   | Service worker: context menu, note storage logic      |
| `content.js`      | Content script: toast notifications on web pages      |
| `content.css`     | Styles for the toast notification overlay             |
| `popup.html`      | Extension popup UI (notes list, search, delete)       |
| `popup.js`        | Popup logic: rendering, search, delete, relative time |
| `create-icons.js` | Node.js script to generate PNG icon files             |
| `icon48.png`      | Extension icon (48x48) -- generated by script         |
| `icon128.png`     | Extension icon (128x128) -- generated by script       |

## How It Works

### Architecture

```
User highlights text and right-clicks
        |
        v
  [Context Menu]  "Save to QuickNote"
        |
        v
  [background.js]  Receives click, creates note object,
                    saves to chrome.storage.local,
                    sends message to content script
        |
        v
  [content.js]     Shows toast notification on page

User clicks extension icon
        |
        v
  [popup.html/js]  Loads notes from chrome.storage.local,
                    renders list, handles search and delete
```

### Key Concepts (Manifest V3)

- **Service Worker** (`background.js`): Replaces the old persistent background page. Runs on-demand and handles events like context menu clicks.
- **Content Scripts** (`content.js`, `content.css`): Injected into every web page to show toast notifications.
- **Popup** (`popup.html`, `popup.js`): The UI that appears when clicking the extension icon in the toolbar.
- **chrome.storage.local**: Persistent key-value storage for the extension's data.

## Troubleshooting

- **"Save to QuickNote" does not appear in context menu**: Make sure you have text selected (highlighted) before right-clicking. The menu item only appears for text selections.
- **Toast notification does not appear**: The content script cannot run on `chrome://` pages, the Chrome Web Store, or other restricted URLs. Try on a regular website.
- **Notes not persisting**: Check that the extension has the `storage` permission in `chrome://extensions/`.
- **Icons missing**: Run `node create-icons.js` to generate the icon files.
