# Day 5 — Capstone

## Project: ShipIt

A full-stack URL shortening service that integrates everything from the bootcamp:

- **Day 1**: REST API design, SQLite database
- **Day 2**: Built using Claude Code
- **Day 3**: PRD-driven development
- **Day 4**: Tests (Vitest), security scan (Semgrep), CI/CD (GitHub Actions), Docker

### What you build
- User authentication (email/password + GitHub OAuth via NextAuth.js)
- URL shortening with custom aliases
- Dashboard for managing your links
- Click analytics with referrer tracking
- Rate limiting
- Docker image + GitHub Actions CI pipeline

### How to work on it
1. Read `PROMPTS.md`
2. Copy the `solution/` scaffolding files (package.json, PRD.md, Dockerfile, etc.) into a new folder
3. Open Claude Code and follow the prompts to build the full application
4. Work through the success criteria in PRD.md when you're done
